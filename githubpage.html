Job Comparison Application

A student nearing graduation needs a tool to evaluate and compare multiple job offers, considering factors beyond just salary, such as benefits and location. To address this need, a straightforward, single-user job comparison application will be developed.

This app was created for the Android platform, adhering to a minimum API level of 28 (Android 9.0 Pie).

Core Features:

Android SQLite Integration: Implemented Android SQLite for storing job details and comparison settings locally on the device. Application is successful in retaining both when reopening until the API is wiped clean.

User Input with EditText: Use EditText elements to allow users to input job information details. CompareJobs Button is also disabled until 2 jobs are entered.

Custom Comparator: Developed a customized comparator class to facilitate the comparison of job offers based on user-defined weights.

Compare Job Selection: Users are able to select two job offers in the compare jobs page and view detailed information for both.
